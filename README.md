# WebPicoPlaca

Para que este aplicativo se ejecute correctamente, se necesita tener instalado lo siguiente:

- Node: 16.14.0
- npm 8.3.1
- angular 12 o 13

# Ejecutar aplicacion web

1. Descargue las dependencias. Dentro de la carpeta del proyecto ejecutar:
- npm install --save

2. Ejecutar la app:
- ng serve

3. Acceder a la URL:
http://localhost:4200/
